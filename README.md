# 알고리즘 챌린지!!

## GREEN
주어진 시간(1시간) 내에 적절하게 해결한 케이스

## YELLOW
주어진 시간(1시간)을 초과하였으나 해결한 케이스
또는 풀이가 너무 비효율적인 케이스

*1~2주 경과 후 풀이를 다시 보지 않고 다시 풀어보기*

## RED
시간을 초과하여도 아예 해결하지 못한 케이스

*풀이 확인 후 1~2시간 경과 후 풀어보고 해결한 경우 YELLOW로 이동*
*그래도 해결 못한 경우 RED에 남겨두고 1~2주 경과 후 다시 풀어보기*
